# hermantrip

Static photo-album site for a 1-year family trip across Nepal, India, Vietnam,
China, Australia, New Zealand, and Thailand. Photos live in a public Google
Drive folder; the site renders them with Hebrew RTL UI at
[hermantrip.tomhe.app](https://hermantrip.tomhe.app).

## Development workflow

This project follows the **cr-tdd-ladder** methodology — every change is a
small milestone PR with FAIL→PASS test evidence, reviewed by a strict CR-ist
subagent, merged with a literal merge commit, tagged `v0.MN`, and archived
in `versions/`. See [docs/cr-rules.md](docs/cr-rules.md) for the rules and
[.claude/agents/crist.md](.claude/agents/crist.md) for the reviewer.

## Layout

- `index.html` — entry point (added in M2).
- `src/lib/` — pure logic, no DOM, no fetch.
- `src/views/` — DOM views.
- `scripts/` — build, test, deploy.
- `data/manifest.json` — generated by `scripts/build-manifest.mjs` (added in M1).
- `docs/` — spec, design language, CR rules, spike outcomes, rebuild guide.
- `versions/` — archived dist tarballs per milestone tag.

## Run tests

```
npm test
```

(Wraps `node --test` — Node 20+ required.)
