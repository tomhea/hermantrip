# Design language

Finalized in M2 when the shell ships. This file is a placeholder so the
CR-ist's R2 anti-AI checklist (which references it) has somewhere to point.

## Will contain (M2)

- Color token definitions (background, surface, text, accent, divider).
- Type scale (Rubik 300/500/700, sizes for display/body/caption).
- Spacing scale (4/8/16/24/32/48).
- Anti-AI checklist (the prose form of the R2 checklist):
  - No purple/indigo/cyan gradient backgrounds.
  - No glassmorphism (`backdrop-filter: blur(...)`).
  - No "Welcome to {site}!" generic hero text.
  - No "Built with..." / "Powered by..." badges.
  - No emoji decorations in UI labels or headings.
  - No AI-illustrated decorative banner art.
  - No multi-color drop shadows.
  - Hebrew copy reads natural — no machine-translated stiffness.

Three principles to design against (from the approved plan):

1. **The photos are the design.** UI chrome recedes — warm paper background,
   single accent color from a real trip photo, hairline dividers, no
   gradients.
2. **Hebrew typography with character.** Rubik family (not Heebo / Noto /
   Assistant which read as AI-default).
3. **Editorial restraint.** Text-first buttons, curated icon handful,
   8 px max grid gutters, no emoji in headings.
